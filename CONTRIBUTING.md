# Contributing to Heavy Space Industry Plan

## Welcome!
Thank you for considering contributing to this open-source vision for humanity's future in space. This is a collaborative project to design a recursive, scalable system for asteroid utilization and space industrialization.

## How to Contribute

### 1. Explore First
- Read the [MANIFESTO](docs/MANIFESTO.md) to understand the vision
- Check the [ROADMAP](ROADMAP.md) for current priorities
- Browse existing designs in the `/design/` folders

### 2. Choose Your Area
- **Mechanical Designs**: CAD models, joint mechanisms, structural analysis
- **Electrical Systems**: Power distribution, sensor networks, motor control
- **Software/AI**: Autonomy algorithms, simulations, control systems
- **Visualization**: 3D models, animations, educational materials
- **Documentation**: Technical specifications, research papers, translations

### 3. Contribution Process
1. **Fork** this repository
2. **Create a branch** for your work: `feature/octopus-hand-improvement` or `docs/gpr-spec`
3. **Add your work** to the appropriate folder
4. **Submit a Pull Request** with a clear description of your changes

### 4. Guidelines
- All contributions must be licensed under AGPLv3.0 (code) or CC-BY-SA (docs/media)
- Use clear file names and include READMEs for complex submissions
- Reference existing issues or create new ones before major work
- Be respectful in discussions - we're building humanity's future together

### 5. Need Help?
- Open an Issue for questions
- Check the [Project Governance](community/PROJECT_GOVERNANCE.md) for decision-making
